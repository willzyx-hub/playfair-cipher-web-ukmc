# Playfair Cipher

This project is web simulation for algorithm Playfair Cipher. 

## Key Features
- Support encryption and decryption Playfair Cipher
- Simulation process every step
- Light and dark mode
- Support two language English and Indonesian

## Tech Stacks
- Next.js
- Tailwind CSS
- TypeScript
